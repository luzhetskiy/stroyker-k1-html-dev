# Установка зависимостей
Для установки зависимостей, выполнить команду в терминале из каталога проекта:
### `npm install`

## Доступные скрипты:

Для запуска проекта в dev режиме:
### `npm start`

Для сборки проекта в production режиме:
### `npm run build`
